# Superstore-Data-Analysis project short explanation 

This project focuses on analyzing the Superstore dataset using Python and Pandas to extract meaningful business insights. The dataset was first loaded and explored to understand its structure, followed by data cleaning to handle missing values and ensure correct data types.

Feature engineering was applied by creating new columns such as Net Sales and Profit Margin to better understand sales efficiency and profitability. The analysis was then performed using groupby operations and pivot tables to summarize sales and profit across categories, regions, and customer segments.

Data visualization techniques such as line charts, bar charts, and pie charts were used to identify patterns and trends over time. The final insights highlight the most profitable categories, regional performance differences, and the relationship between discounts, sales, and profit. Overall, the project demonstrates how data analysis can support better business decision-making.
